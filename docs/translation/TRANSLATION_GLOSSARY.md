# Greenshot Translation Glossary

This glossary maintains consistent translations of Greenshot-specific terms across all languages. When translating, always refer to this glossary to ensure consistency.

## How to Use This Glossary

1. **Before translating**: Check if the term appears in this glossary
2. **Use the approved translation**: Apply the glossary term consistently
3. **Add new terms**: When translating important recurring terms, add them here with context
4. **Update when needed**: If a better translation is found, update the glossary and note the change

## Format

Each entry should include:
- **English term** (as used in source)
- **Context** (where/how it's used)
- **Translation per language**
- **Notes** (special considerations)

## Special Formatting Notes

### Keyboard Shortcuts

**Policy**: Keep the shortcut letter in parentheses but translate the label

Examples:
- English: "Crop (C)"
- German: "Zuschneiden (C)"
- French: "Recadrer (C)"

### Placeholders

**Policy**: Keep placeholders (`{0}`, `{1}`, etc.) in the same position and format

Examples:
- English: "Export failed for {0}"
- German: "Export nach '{0}' fehlgeschlagen"
- Note: Placeholder position may change to match grammar

### HTML Content

**Policy**: Keep HTML tags unchanged, translate only text content

Example:
```xml
<!-- English -->
<resource name="about_license">Copyright © 2004-2026 Thomas Braun, Jens Klingen, Robin Krom
Greenshot comes with ABSOLUTELY NO WARRANTY. This is free software...</resource>

<!-- German -->
<resource name="about_license">Copyright © 2004-2026 Thomas Braun, Jens Klingen, Robin Krom
Für Greenshot besteht KEINERLEI GARANTIE. Greenshot ist freie Software...</resource>
```

## Language-Specific Style Guides

### Formality

- **German**: Use formal "Sie" form
- **French**: Use formal "vous" form  
- **Spanish**: Use "usted" or informal "tú" depending on convention
- **Portuguese**: Generally formal

### Capitalization

- **English**: Title Case for menu items, Sentence case for descriptions
- **German**: Capitalize all nouns
- **French**: Capitalize first word and proper nouns
- **Others**: Follow language conventions
